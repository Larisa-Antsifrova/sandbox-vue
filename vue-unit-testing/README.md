# Unit Testing

## Steps

1. Create a test suite

```
describe(...)
```

2. Set up the tests

```
test(...)
```

3. Mount the component with vue-test-utils

```
mount(...)
```

4. Set data, if necessary

```
setData(...)
```

5. Asert what the result should be

```
expect(...)
```
